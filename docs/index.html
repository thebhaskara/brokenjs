Model

following functions are included

Attributes

initializes internal attributes object by the object provided,otherwise an empty object is initialized.

get

Get value from attributes, at the path provided.
Internally uses lodash _.get

{String} [path] - string object path.
attributesObj.get('employee.name'); // returns 'Rayn'attributesObj.get('employee.id'); // returns 10

set

Set value to attributes, at the path provided.
Internally uses lodash _.set

{String} path - string object path.
{Object} value - value to be stored at the provided path.
attributesObj.set('employee', { id:10, name: 'Ryan' })

destroy

Deletes internal attributes instance.

undefined
attributesObj.destroy();

Destroy

This class adds the ability to destroy object properly.
This provides onDestroy(Callbacks) property internally.onDestroy callbacks are run on the destroy function call.
This is designed to be used to destory object when needed,thus preventing memory leaks
